# MinSU Dashboard Image Setup Instructions

## Required Images

Please save the following images to the `public/images/` directory:

### 1. Campus Background Images

Save the campus photos you provided as:

- **`minsu-campus-1.jpg`** - Main building with green overlay (first image)
- **`minsu-campus-2.jpg`** - Building at sunset (second image)
- **`minsu-campus-3.jpg`** - Campus grounds with green field (third image)
- **`minsu-campus-4.jpg`** - Building front view (fourth image or any additional campus photo)

### 2. University Logo

Save the MinSU seal/logo as:

- **`minsu-logo.png`** - The official Mindoro State University circular logo with the donkey emblem

## File Locations

All images should be placed in:
```
public/images/
├── minsu-campus-1.jpg
├── minsu-campus-2.jpg
├── minsu-campus-3.jpg
├── minsu-campus-4.jpg
└── minsu-logo.png
```

## Image Specifications

- **Campus Images**: Use high-resolution photos (1920x1080 or higher recommended)
- **Logo**: Use PNG format with transparent background if possible (512x512 or higher)

## After Adding Images

1. Clear browser cache or do a hard refresh (Ctrl+F5)
2. The dashboard will automatically display the campus images as backgrounds
3. Each system slide will show a different campus photo
4. The logo will appear in the header next to "Mindoro State University"
